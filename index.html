About me

Hi I am Johannes, passionated Frontend Developer, born and raised in Berlin, Germany.

I create web frontend with well structred HTML, responsive CSS and reusable JavaScript.

For my work I use React, Redux, Vue.js, jQuery, lodash, D3.js, webpack, Node.js, Less/Sass, SVG, etc. I also know how to use Bash shell and git.

I like to work on a Unix-like operating system like MacOS but as a fan of free software, I am also using GNU/Linux systems with the Gnome Desktop.

Currently I am working for Das Büro am Draht in Berlin.
